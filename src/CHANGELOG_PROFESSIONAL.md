# سجل ترقية فزاع الاحترافية

## الإصدار 1 — التشغيل والتحصيل والثقة الاستثمارية

### واجهات جديدة
- `CollectionCenterView.tsx`
- `InvestorPortalView.tsx`
- `FairDistributionView.tsx`
- `ApprovalCenterView.tsx`

### بنية البيانات
- العملاء وعقود التقسيط والأقساط.
- معاملات دفع غير قابلة للضياع وتدعم التجزئة.
- أنشطة التحصيل ووعود السداد.
- طلبات الاعتماد.
- سجل قرارات التوزيع.
- مركز الإشعارات.

### أداء وتجربة مستخدم
- تحميل الشاشات عند الطلب.
- تقسيم مكتبات React وSupabase والرسوم والحركات إلى حزم مستقلة.
- خفض ملف التطبيق الرئيسي من أكثر من 500KB إلى نحو 21KB قبل gzip، مع فصل المكتبات.
- شاشة خطأ مستقلة لكل وحدة.
- قائمة جانبية مصنفة حسب الرئيسية والعمليات والمالية والإدارة.
